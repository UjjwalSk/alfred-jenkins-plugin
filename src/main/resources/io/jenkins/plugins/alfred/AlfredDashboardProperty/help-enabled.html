Enable the Alfred Dashboard to display aggregated job statistics at the top of this view.

The dashboard shows:

  • Total number of jobs in the view
  • Count of successful, failed, and unstable jobs
  • Aggregated test failure analysis with categorization
  • Top failure categories across all jobs

Configuration via Job DSL: 

listView('my-view') {
  properties {
    alfredDashboard {
      enabled(true)
    }
  }
}
